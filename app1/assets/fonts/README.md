# Fonts Required by FORGE

Download these fonts from Google Fonts and place the .ttf files in this directory:

1. **BebasNeue-Regular.ttf** - https://fonts.google.com/specimen/Bebas+Neue
2. **ShareTechMono-Regular.ttf** - https://fonts.google.com/specimen/Share+Tech+Mono
3. **BarlowCondensed-Regular.ttf** - https://fonts.google.com/specimen/Barlow+Condensed
4. **BarlowCondensed-Medium.ttf** - https://fonts.google.com/specimen/Barlow+Condensed
5. **BarlowCondensed-SemiBold.ttf** - https://fonts.google.com/specimen/Barlow+Condensed
6. **BarlowCondensed-Bold.ttf** - https://fonts.google.com/specimen/Barlow+Condensed

All fonts are free under OFL license.
